# Discord案内動画 撮影指示書

> 形式：画面録画 + テキスト追加（音声なし）
> 想定時間：2〜3分
> 対象：海外の新規参加者

---

## 事前準備

### Discordの準備
- [ ] 価格表チャンネルに最新の価格表を投稿しておく
- [ ] デリバリーレビューチャンネルにレビューがある状態にする
- [ ] チケット用のボタンが表示されている状態にする
- [ ] テスト用のチケットを1つ作成しておく（注文例を表示するため）

### 画面の準備
- [ ] Discordをフルスクリーンまたはウィンドウ最大化
- [ ] 通知をオフにする（録画中に通知が出ないように）
- [ ] 不要なチャンネルやDMは非表示にする
- [ ] ダークモード推奨（見やすい）

### 録画設定
- [ ] 解像度：1920x1080（フルHD）推奨
- [ ] フレームレート：30fps
- [ ] マウスカーソルを表示する設定にする

---

## 撮影シーン一覧

| シーン | 秒数 | 画面 | 操作 |
|--------|------|------|------|
| 1 | 5秒 | サーバートップ | 静止 |
| 2 | 10秒 | 価格表チャンネル | スクロール |
| 3 | 15秒 | チケットボタン | クリック |
| 4 | 10秒 | チケットチャット | 静止→入力例 |
| 5 | 5秒 | テキスト画面 | 静止 |
| 6 | 5秒 | テキスト画面 | 静止 |
| 7 | 5秒 | テキスト画面 | 静止 |
| 8 | 10秒 | デリバリーレビュー | スクロール |
| 9 | 5秒 | チケットボタン | 静止 |

## シーン別 撮影詳細

---

### シーン1：オープニング（5秒）

**画面構成**
```
┌─────────────────────────────────┐
│  Discordサーバーのトップ画面    │
│  （サーバー名・アイコンが見える）│
│                                 │
│                                 │
└─────────────────────────────────┘
```

**操作**
- サーバーのトップ（#welcomeなど）を表示
- 動かさず静止

**追加テキスト（英語）**
```
Welcome to our Discord server!
```

**追加テキスト（日本語訳）**
```
私たちのDiscordサーバーへようこそ！
```

**テキスト位置**: 画面中央下部

---

### シーン2：価格表チャンネル（10秒）

**画面構成**
```
┌─────────────────────────────────┐
│ #price-list                     │
│ ┌─────────────────────────────┐ │
│ │ 価格表の投稿                │ │
│ │ （商品名・価格が見える）    │ │
│ │                             │ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

**操作**
1. 価格表チャンネルをクリック
2. 価格表の投稿をゆっくりスクロールして見せる

**追加テキスト（英語）**
```
Every day, we post updated price lists here.
Check this channel for the latest prices.
```

**追加テキスト（日本語訳）**
```
毎日、最新の価格表をここに投稿しています。
このチャンネルで最新価格をご確認ください。
```

**テキスト位置**: 画面上部（価格表の邪魔にならない位置）

---

### シーン3：チケット作成（15秒）

**画面構成 - 前半**
```
┌─────────────────────────────────┐
│ #order または #ticket           │
│                                 │
│    ┌─────────────────┐          │
│    │  Open Ticket    │ ← ここを │
│    │     ボタン      │   クリック│
│    └─────────────────┘          │
│                                 │
└─────────────────────────────────┘
```

**操作（前半）**
1. チケットボタンがあるチャンネルを表示
2. マウスをボタンに移動
3. クリック

**追加テキスト（英語）- 前半**
```
To place an order, click the "Open Ticket" button.
```

**追加テキスト（日本語訳）- 前半**
```
注文するには「Open Ticket」ボタンをクリック。
```

---

**画面構成 - 後半**
```
┌─────────────────────────────────┐
│ ticket-xxxx（新しく作成された） │
│                                 │
│  「チケットが作成されました」   │
│  的なメッセージが表示される     │
│                                 │
└─────────────────────────────────┘
```

**操作（後半）**
- チケットが作成されて開くのを待つ
- 少し静止

**追加テキスト（英語）- 後半**
```
This creates a private chat just for you.
A dedicated sales rep will assist you here.
```

**追加テキスト（日本語訳）- 後半**
```
あなた専用のプライベートチャットが作成されます。
専任の販売担当者が対応いたします。
```

**追加テキスト（日本語訳）**

### シーン5：送料・発送方法（5秒）

**画面構成**
```
┌─────────────────────────────────┐
│                                 │
│   ┌─────────────────────────┐   │
│   │  Shipping: ¥5,000       │   │
│   │  (flat rate)            │   │
│   │                         │   │
│   │  Carriers:              │   │
│   │  FedEx / DHL            │   │
```

- 画面録画を一時停止し、テキストのみの画面を挿入

**追加テキスト（英語）**
```

```

**追加テキスト（日本語訳）**
```
送料：一律5,000円
配送：FedEx / DHL
```

**背景色**: シンプルな単色（#2C2F33 など Discord風の色）

## 編集時の注意点

### テキストのスタイル
- フォント：読みやすいサンセリフ体（Noto Sans, Arial など）
- サイズ：大きめ（最低24px以上）
- 色：白文字 + 黒の縁取りまたは半透明の背景
- 表示位置：画面の重要な部分を隠さない

### トランジション
- シーン間はフェードまたはカット
- 派手なエフェクトは不要

### BGM（任意）
- 静かで邪魔にならない音楽
- 著作権フリーのものを使用
