# 国別ルール早見表

| 国 | 業者 | 金額上限 | 関税 |

| 🇺🇸 アメリカ | **FedEx** | 2,500 USD | 16% |

---

# 🇦🇺 オーストラリア

| 項目 | ルール |
|------|--------|
| 推奨業者 | **DHL** |
| 金額上限 | **1梱包 1,000 AUD以下** |
| 理由 | 1,000AUD以下は**関税免除** |

### ポイント

- 必ず1,000AUD以下になるよう**分割**
- 顧客にとって関税がかからずお得
- 複数梱包になる場合は事前に顧客に確認

---

# 🇸🇬 シンガポール

| 項目 | ルール |
|------|--------|
| 推奨業者 | DHL / FedEx |
| 金額上限 | **希望があれば 400 SGD以下** |
| 理由 | 400SGD以下は**関税免除** |

### ポイント

- 顧客から希望があれば分割対応
- 利益と工数を考慮して判断
- 希望がなければ通常発送でOK

---

# 🇺🇸 アメリカ（金額）

| 項目 | ルール |
|------|--------|
| 推奨業者 | **FedEx** |
| 金額上限 | **1梱包 2,500 USD以下** |
| 理由 | 2,500USD超は**通関で止まるリスク高** |

### ポイント

- 2,500USD超の場合は**分割必須**
- 顧客に事前確認「分割しますか？」

---

# 🇺🇸 アメリカ（関税対応）

| 項目 | 内容 |
|------|------|
| 関税率 | **約16%** |
| 対応方法 | **事前徴収 + DDP設定** |

### 手順

```
1. 見積もり時に関税16%を上乗せして請求
2. FedExラベル作成時に「関税支払い」を設定
   └「受取人払い」→「発送人払い（マイアカウント）」に変更
3. こちらで関税を支払うDDP形式
```

---

# DDP対応のメリット

| メリット |
|----------|
| 顧客が通関時に支払う手間なし |
| 受取拒否リスクが減る |
| 顧客満足度UP |

> DDP = Delivered Duty Paid（関税込み配送）

---

# ⚠️ 絶対NGな対応

<div class="warning">

## アンダーバリュー

実際より低い価格での申告 → **違法**

## ギフト扱いでの送付

商業目的なので → **NG**

</div>

# 価格申告について

## ルール

- **必ず実際の取引価格を正確に申告**
- 顧客から「安く書いて」と言われても断る

## 断り方

> 「税関リスクがあるため対応できません」

---

# よくあるトラブル

| トラブル | 対応 |
|----------|------|
| 住所不備で返送 | 顧客に正確な住所を再確認 |
| 税関で止まった | 顧客に追加書類の提出を依頼 |
| 紛失・破損 | 追跡番号で調査、保険対応 |

> 配送トラブルは早めに報告
