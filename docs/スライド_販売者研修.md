# 1. 顧客対応の基本

---

## レスポンスが命

| 目標 | 詳細 |
|------|------|
| **即レス** | 努力目標 |
| 実運用 | なるべく早く |

### なぜ？

- 返信が早いほど **成約率UP**
- 顧客も「やり取りが早い・楽」で **常連化しやすい**

---

## 対応時間

| 項目 | ルール |
|------|--------|
| 対応時間 | **実質24時間** |
| 理由 | 海外顧客との時差 |

> 💡 夜間対応は強制ではないが、早ければ早いほど機会損失を防げる

---

## コミュニケーションのトーン

### OK

- 敬語、カジュアル、絵文字 → 全てOK
- 翻訳やAIを使うので顧客に合わせて柔軟に

### NGライン

- 失礼、取引相手として信用できない、と思われる対応

# 2. NGルール

---

## 絶対NGワード

| NG | 理由 |
|----|------|
| 人種差別的な発言 | **絶対NG** |
| 相手を否定する言葉 | 信頼関係を壊す |

---

## 絶対NG対応

| NG | 内容 |
|----|------|
| **アンダーバリュー** | 実際より低い価格での申告 |
| **ギフト扱いでの送付** | 税金逃れの手法 |

### ⚠️ 顧客から依頼されても断ること

> 理由：「税関リスクがあるため対応できません」と伝える

---

## クレーム対応

| 対応 | 権限 |
|------|------|
| 「次回値引きするよ」程度 | ✅ **各自判断OK** |
| 返金・返品対応 | ⚠️ **要相談** |
| トラブル拡大の可能性 | 🚨 **即相談** |

> 💡 クレームは早めに相談。ブランディングを守る。

# 3. 見積もり・価格

---

## 見積もりフォーマット（統一）

```
【見積もり】
━━━━━━━━━━━━━━
商品名：[商品名]
数量：[個数]
単価：$[価格]
小計：$[合計]
━━━━━━━━━━━━━━
送料：$50（一律）
━━━━━━━━━━━━━━
合計：$[総額]
━━━━━━━━━━━━━━
有効期限：[明日の日付]まで
決済方法：Wise推奨（手数料0）
```

---

## 価格決定のルール

| 項目 | ルール |
|------|--------|
| 基本価格 | N&Uが提示 |
| 値下げ交渉 | 各自判断でOK |
| 目標粗利率 | **6.5%** |
| 報酬 | 粗利の20% |

> 💡 還付金（10%）があるので、仕入れ値より安くてもOK

---

## 送料・決済

### 送料

| ルール |
|--------|
| **一律 $50（約5,000円）** |

### 決済方法

| 方法 | 手数料 | 推奨 |
|------|--------|------|
| **Wise** | **0%** | ⭐ 最推奨 |
| PayPal | 5% | 可 |

---

## 有効期限

| ルール |
|--------|
| **翌日中まで** |

### 理由

- 相場変動が毎日ある
- 事故防止のため翌日中が限界

# 4. 日次業務フロー

---

## 1日のスケジュール

### 朝〜昼

- [ ] 前日夜の問い合わせに返信
- [ ] チケット対応
- [ ] 個別チャットでの見積もり対応

### 夕方（価格更新後）

- [ ] 新しい価格表を確認
- [ ] **担当顧客に価格更新を連絡**
- [ ] 処分対象商品を優先的に案内

---

## 価格更新の連絡

### トリガー

> N&Uが価格表を更新したら → 担当顧客に連絡

### なぜ？

- 早ければ早いほど成約機会が増える
- 競合より先に連絡することが重要

---

## 担当顧客の目安

| 項目 | 目安 |
|------|------|
| 1人あたり担当顧客数 | **約400名** |
| 連絡停止の基準 | 1ヶ月以上返信なし |

> 全員がVIPではないので対応可能

# 5. Discord運用

---

## 新規参加者の流れ

```
1. SNS等から招待リンクをクリック
2. サーバーに参加
3. 参加時アンケートに回答
4. ロールが自動付与
5. 案内動画を視聴
6. チケットを開封して個別対応開始
```

---

## 担当の振り分け

| 参加経路 | 担当 |
|----------|------|
| 販売者Aの招待リンク | 販売者A |
| 販売者Bの招待リンク | 販売者B |
| 誰経由でもない | ランダム |

> mee6のInvite機能で自動計測

---

## チケットシステム

1. 新規参加者が案内動画を視聴
2. 「オープンチケット」をタップ
3. 個別チャンネルが自動作成
4. 担当販売者が対応

> 動画を見ていない人には「まず動画を見て」と案内

---

## デリバリーレビュー

### 目的

- 取引実績を可視化
- 新規顧客への信頼構築

### 依頼方法

```
商品の到着おめでとうございます！🎉
#デリバリーレビュー に感想を投稿いただけると嬉しいです！
```

# 6. SNS更新

---

## 投稿の3段階

| レベル | 内容 | 頻度 |
|--------|------|------|
| 1 | 在庫写真（タイムスタンプ付き） | 毎日1回以上 |
| 2 | 開封動画、インパクト強め | 週3回 |
| 3 | 顔出し喋り、信頼性訴求 | 週3回に含める |

| 曜日 | 投稿内容 |

| 火 | **作り込み**（開封動画） |
| 水 | 在庫写真 |
| 木 | **作り込み**（顔出し喋り） |
| 金 | 在庫写真 |
| 土 | **作り込み**（信頼性訴求） |

## 報酬計算例

```
報酬 = 粗利 × 20%

例）月商1,500万、粗利率6.5%の場合
粗利 = 1,500万 × 6.5% = 97.5万
報酬 = 97.5万 × 20% = 19.5万
```
