# 1. 事業の全体像

---

## このビジネスは何をするのか

日本国内でトレーディングカードを仕入れ、海外の顧客に販売します。

| 項目 | 内容 |
|------|------|
| 取扱商品 | ポケモンカード、遊戯王、ワンピースなど |
| 顧客 | 海外のバイヤー、コレクター、ショップオーナー |
| 販売チャネル | Discord、Instagram |

---

## 役割分担

| 担当 | 業務内容 |
|------|----------|
| N&U（提携先） | 仕入れ、在庫管理、梱包、発送、価格表作成 |
| クレイ（販売者） | 顧客対応、見積もり作成、発送ラベル作成 |
| 山崎 | 全体統括、資金管理、最終判断 |

販売者は「売る」ことに集中できる体制です。

---

## 販売者の報酬

| 項目 | 内容 |
|------|------|
| 報酬 | 粗利の20% |
| 締め日 | 翌月末 |
| 支払日 | 翌々月末 |

> 送料が月末に確定するため、このサイクルになっています。先払いも可能です。

# 2. 利益の仕組み

---

## 通常の物販の利益

```
売上 − 仕入れ − 経費 = 利益
```

仕入れ値より高く売らないと利益が出ません。

薄利多売か、高く売るかの勝負になります。

---

## この事業の利益

```
売上 − 仕入れ − 経費 + 消費税還付金 = 利益
```

**還付金があるため、仕入れ値と同額で売っても利益が出ます。**

これが最大の競争優位性です。

---

## 消費税還付とは

日本の消費税は「国内消費」に対する税金です。

海外に輸出する場合、消費されるのは海外なので、日本で支払った消費税は還付されます。

これを「輸出免税」と呼びます。

---

## 還付金の計算

仕入れ時に支払った消費税が戻ってきます。

```
還付金 = 仕入れ値 ÷ 11
```

| 仕入れ値 | 還付金 |
|----------|--------|
| 110,000円 | 10,000円 |
| 55,000円 | 5,000円 |
| 220,000円 | 20,000円 |

---

## 還付金を受け取るための条件

還付金は自動で戻ってくるわけではありません。輸出の証拠と取引記録が必要です。

| 必要書類 | 内容 |
|----------|------|
| 輸出許可証 | 税関で輸出が許可された証明 |
| 発送ラベル（控え） | 輸出した証拠 |
| インボイス（請求書） | 取引内容の証明 |
| 取引記録 | 売上と仕入れの対応関係 |
| 入金記録 | 代金を受け取った証明 |

---

## 還付が否認されるケース

| リスク | 原因 |
|--------|------|
| 輸出の証拠がない | ラベル控えや許可証を保管していない |
| 取引記録が不明確 | どの仕入れがどの販売に対応するか不明 |
| 不正申告の疑い | アンダーバリューなど |

**だから、ラベル管理・請求書管理・取引管理が重要なのです。**

---

# 3. 絶対NG事項

---

## アンダーバリューの禁止

### アンダーバリューとは

実際の取引価格より低い金額を発送ラベルに記載することです。

| 実際の販売価格 | ラベル記載額 | 判定 |
|----------------|--------------|------|
| 150,000円 | 150,000円 | OK |
| 150,000円 | 75,000円 | NG（アンダーバリュー） |

---

## なぜアンダーバリューが禁止なのか

| 理由 | 詳細 |
|------|------|
| 違法行為 | 関税法違反、脱税幇助に該当 |
| 還付金が否認される | 輸出記録と売上が合わず申請が通らない |
| 税関で止まる | 不自然な申告は調査対象になる |
| 会社全体のリスク | 一件でも問題があると全取引が調査対象に |

顧客から依頼されても断ってください。

> 「税関リスクがあるため対応できません」と伝えてください。

---

## ギフト扱いの禁止

### ギフト扱いとは

商業取引なのに「ギフト（贈り物）」として発送することです。

### なぜ禁止なのか

| 理由 | 詳細 |
|------|------|
| 違法 | 商業取引は商業申告が必須 |
| 詐欺 | 関税詐欺に該当する |
| 保険が効かない | ギフト扱いだと商業保険が適用されない |

---

## その他の禁止事項

| 禁止事項 | 理由 |
|----------|------|
| 人種差別的な発言 | 絶対禁止。会社の信用に関わる |
| 相手を否定する言葉 | 関係悪化、クレームの原因 |

### クレーム対応の判断基準

| 対応内容 | 判断 |
|----------|------|
| 次回の値引き程度 | 各自の判断で可 |
| 返金・返品 | 相談してください |
| トラブル拡大の可能性 | すぐに相談してください |

---

# 4. 価格と見積もり

---

## 決済方法と手数料

| 決済方法 | 手数料 | 備考 |
|----------|--------|------|
| **Wise** | **0%** | 優先して案内 |
| PayPal | 5% | 入金全額にかかる |

> PayPalは商品代金だけでなく、送料込みの入金全額に対して5%かかります。

基本は**Wiseを案内**してください。

---

## 送料の仕組み

| 項目 | 金額 |
|------|------|
| 案内上の送料 | **一律5,000円** |
| 実際の送料 | 8,000〜15,000円が多い |
| 差額 | こちらで負担 |

売りやすく、案内しやすくするために統一しています。

---

## 見積書でのディスカウント表示

見積書に重さと国を入力すると、本来の送料が自動計算されます。

**必ずディスカウント欄を使ってください。**

| 本来の送料 | 案内送料 | ディスカウント表示 |
|------------|----------|-------------------|
| 12,000円 | 5,000円 | −7,000円 |
| 15,000円 | 5,000円 | −10,000円 |
| 12,000円 | 0円（無料） | −12,000円 |

**頑張って値引きしていることを視覚的に見せて、恩を売る。**

# 5. 利益計算の実例

---

## 利益計算の基本式

```
粗利 = 入金額 − 実送料 − 仕入れ値 − 手数料 + 還付金
```

ポイントは「実送料」と「還付金」です。

案内上の送料（5,000円）ではなく、実際にかかる送料で計算します。

---

## 実例①：通常パターン

| 項目 | 金額 |
|------|------|
| 仕入れ値 | 110,000円 |
| 販売価格 | 150,000円 |
| 案内送料 | 5,000円 |
| 実送料 | 12,000円 |
| 決済 | Wise |

顧客からの入金：155,000円（商品＋送料）

---

## 実例①：計算結果

| 項目 | 金額 |
|------|------|
| 入金額 | +155,000円 |
| 実送料 | −12,000円 |
| 仕入れ値 | −110,000円 |
| **現金収支** | **+33,000円** |
| 還付金 | +10,000円 |
| **粗利** | **+43,000円** |

見積書には「送料ディスカウント：−7,000円」と表示して恩を売る。

---

## 実例②：送料無料にした場合

| 項目 | 金額 |
|------|------|
| 仕入れ値 | 110,000円 |
| 販売価格 | 120,000円 |
| 案内送料 | 0円（無料） |
| 実送料 | 12,000円（全額負担） |
| 決済 | Wise |

顧客からの入金：120,000円

---

## 実例②：計算結果

| 項目 | 金額 |
|------|------|
| 入金額 | +120,000円 |
| 実送料 | −12,000円 |
| 仕入れ値 | −110,000円 |
| **現金収支** | **−2,000円** |
| 還付金 | +10,000円 |
| **粗利** | **+8,000円** |

**現金収支はマイナスでも、還付金で利益が出る。**

見積書には「送料ディスカウント：−12,000円（送料無料）」と表示して最大限の恩を売る。

---

## 実例③：仕入れ値以下で売った場合

| 項目 | 金額 |
|------|------|
| 仕入れ値 | 110,000円 |
| 販売価格 | 105,000円 |
| 送料 | 5,000円 |
| 決済 | Wise |

顧客からの入金：110,000円

---

## 実例③：計算結果

| 項目 | 金額 |
|------|------|
| 入金額 | +110,000円 |
| 送料 | −5,000円 |
| 仕入れ値 | −110,000円 |
| **現金収支** | **−5,000円** |
| 還付金 | +10,000円 |
| **粗利** | **+5,000円** |

仕入れ値以下で売っても利益が出る。在庫処分や関係構築に活用。

---

## 実例④：PayPal決済の場合

| 項目 | 金額 |
|------|------|
| 仕入れ値 | 110,000円 |
| 販売価格 | 150,000円 |
| 送料 | 5,000円 |
| 決済 | PayPal（手数料5%） |

顧客からの入金：155,000円
PayPal手数料：155,000円 × 5% = 7,750円

---

## 実例④：計算結果

| 項目 | 金額 |
|------|------|
| 入金額 | +155,000円 |
| PayPal手数料 | −7,750円 |
| 実送料 | −12,000円 |
| 仕入れ値 | −110,000円 |
| **現金収支** | **+25,250円** |
| 還付金 | +10,000円 |
| **粗利** | **+35,250円** |

Wiseの場合（43,000円）と比較して**7,750円少なくなる**。

# 6. 国別発送ルール

---

## 国別ルール一覧

| 国 | 配送業者 | 金額上限 | 上限を超えると |
|----|----------|----------|----------------|
| オーストラリア | DHL | 1,000 AUD | 関税発生 |
| シンガポール | どちらでも | 400 SGD | 関税発生 |
| アメリカ | FedEx | 2,500 USD | 通関で止まるリスク |

---

## オーストラリア

| 項目 | 内容 |
|------|------|
| 配送業者 | **DHL** |
| 金額上限 | **1梱包 1,000 AUD以下** |
| 理由 | 1,000 AUD以下は関税免除 |

**必ず1,000 AUD以下になるよう分割してください。**

複数梱包になる場合は事前に顧客へ確認。

---

## シンガポール

| 項目 | 内容 |
|------|------|
| 配送業者 | DHLまたはFedEx |
| 金額上限 | **希望があれば 400 SGD以下** |
| 理由 | 400 SGD以下は関税免除 |

顧客から希望があれば分割対応。希望がなければ通常発送でOK。

---

## アメリカ

| 項目 | 内容 |
|------|------|
| 配送業者 | **FedEx** |
| 金額上限 | **1梱包 2,500 USD以下** |
| 関税率 | **約16%** |

2,500 USDを超えると通関で止まるリスクが高い。超える場合は分割必須。

---

## アメリカの関税対応（DDP）

DDP = Delivered Duty Paid（関税込み配送）

### 手順

1. 見積もり時に関税16%を上乗せして請求
2. FedExラベル作成時に「関税支払い」を設定
3. 「受取人払い」→「発送人払い（マイアカウント）」に変更

### メリット

- 顧客が通関時に支払う手間がなくなる
- 受取拒否のリスクが減る
- 顧客満足度が上がる

# 7. 顧客対応の考え方

---

## まず顧客の立場で考える

あなたは海外のサイトや、知らない個人・会社から買い物をするとき、怖くないですか？

- 本当に届くのか？
- 詐欺じゃないか？
- 何かあったとき対応してくれるのか？

**顧客も同じ不安を抱えています。**

---

## 信頼が全ての基盤

| 施策 | なぜ効くのか |
|------|-------------|
| 顔出し | 「この人から買っている」という安心感 |
| 丁寧な説明 | 「ちゃんとした業者だ」という印象 |
| デリバリーレビュー | 「他の人も届いている」という証拠 |
| 口コミ | 「みんなが使っている」という社会的証明 |

これら全ては「**信頼を得る**」ためにやっています。

---

## 顧客のために何ができるか？

信頼を得るために、顧客のために何ができるかを常に考えてください。

- 不安を減らすために何ができるか？
- スムーズに取引するために何ができるか？
- また買いたいと思ってもらうために何ができるか？

**この視点が全ての行動の基準になります。**

---

## 成約を優先する考え方

### 大前提

粗利は残さないと報酬が生まれません。

**しかし、買うつもりで問い合わせしている顧客に対しては、売ることを優先してください。**

---

## なぜ成約を優先するのか

特に初回購入が重要です。

うちで買わなかった場合、その顧客は**大体ほかの販売者から購入しています**。

| 購入先を決める要素 |
|-------------------|
| 価格 |
| レスの速さ |
| やり取りのスムーズさ |
| 安心感 |

これらが複合的に絡んでいます。

---

## 薄利でも売る理由

薄利だからといって売らないことは、**のちの利益を捨てること**になります。

| ❌ 目先の利益で判断 | ⭕ 長期の利益で判断 |
|-------------------|-------------------|
| 粗利が少ないから売らない | 薄利でも成約させる |
| 1回の取引で評価 | 1ヶ月単位でその顧客が生み出す利益を見る |
| 顧客を逃す | リピーターになる |

**目先の損は気にしない。基本、取引は成約させることを優先。**

---

## 基本姿勢

海外の顧客も同じ人間です。

日本人でも良い人・悪い人がいるように、海外の方も同じです。

**国籍で判断せず、個人として向き合ってください。**

---

## 取引の基本ルール

| 項目 | ルール |
|------|--------|
| 決済 | **顧客の先入金が基本** |
| 為替 | Wise経由のため為替差損なし |
| 粗利率5%以下 | 相談してください |
| クレーム | 都度相談してください |
| 判断に迷うこと | 気軽に聞いてください |

---

## 売上目標

| 項目 | 内容 |
|------|------|
| 目標 | 月商1,000万円以上 |
| ノルマ | なし（強制ではない） |
| 未達時 | 行動目標を一緒に作る |

**やるべきことをやっていれば、利益は出ます。**

---

## 緊急時の連絡

| 状況 | 連絡先 |
|------|--------|
| 緊急時 | 山崎に電話（起きていれば即対応、出られなければ折り返す） |
| 通常連絡 | LINE |

不安なことがあれば、何でも聞いてください。

---

## 値引きの基本的な考え方

このビジネスのゴールは「**全体粗利総額を伸ばすこと**」です。

1回の取引ではなく、**その顧客との取引全体**で利益を見ます。

| 視点 | 考え方 |
|------|--------|
| 1回目 | 薄利でも成約優先。信頼を築く |
| 2回目以降 | 関係ができているので利益を乗せやすい |
| 長期 | リピートで利益が積み上がる |

**利益はタイミング次第で出せます。**

---

## 値引きの伝え方

ただ安くするのではなく、**恩を売る意識**が大切です。

「あなたのために特別に対応します」と伝えてください。

### 仕入れ値の開示も有効

> 「正直に言うと、仕入れ値は110,000円です。この価格だと利益はほぼありませんが、今回は特別に対応します。」

透明性が信頼につながり、過度な値引き交渉の抑制にもなります。

---

## 値下げ交渉への対処

海外では値下げ交渉が当たり前の文化があります。やり取りが負担なら、**個別ルールを決めてしまう**方法が有効です。

### 個別ルールの例

| ルール | 内容 |
|--------|------|
| 1箱1万円以上 | 値下げ余地あり |
| 1万円以下 | 値下げ不可 |
| 発注30万円以上 | 送料無料 |
| 年間1億円以上の常連 | 毎回3%オフ固定 |

やり取りが楽になる工夫は、個別にどんどん取り入れてください。

---

## 目指す関係性

「交渉」ではなく「**相談**」ベースで話が来る関係を目指します。

- 「この供給者ならスムーズにやり取りできる」
- 「こちらの事情もわかってくれている」

このような関係ができると、顧客は離れにくくなります。

---

## 価格以外で恩を売る方法

| 方法 | 具体例 |
|------|--------|
