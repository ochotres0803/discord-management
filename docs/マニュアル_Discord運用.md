# Discord運用マニュアル

> 最終更新：2026年2月25日

---

## 1. 使用ツール

| ツール | 用途 |
|--------|------|
| **mee6（プレミアム）** | Bot機能全般 |

### ■ mee6で実装している機能

- チケットシステム
- 招待トラッキング（Invite）
- ロール自動付与
- ウェルカムメッセージ

---

## 2. 新規参加者の流れ

### ■ フロー

```
1. SNS等から招待リンクをクリック
2. サーバーに参加
3. 参加時アンケートに回答
   └ 「誰の招待で来ましたか？」
   └ 「あなたの属性は？」
4. ロールが自動付与
5. 案内動画を視聴
6. チケットを開封して個別対応開始
```

---

## 3. ロール付与ルール

### ■ 招待元による担当振り分け

| 参加経路 | ロール付与 | 担当 |
|----------|-----------|------|
| 販売者Aの招待リンク | 販売者Aのロール | 販売者A |
| 販売者Bの招待リンク | 販売者Bのロール | 販売者B |
| 誰経由でもない（広告、紹介等） | **ランダム付与** | ランダム |

### ■ 顧客属性ロール

参加時のアンケートで回答：

| 属性 | 説明 |
|------|------|
| 転売ヤー | 仕入れ目的 |
| コレクター | コレクション目的 |
| ショップオーナー | 店舗仕入れ |
| その他 | 上記以外 |

※現状、属性ロールの効果は限定的

---

## 4. チケットシステム

### ■ 開封方法

```
・「オープンチケット」ボタンをタップ
・mee6で自動生成されたチャンネルで個別対応
```

### ■ 対応の流れ

```
1. 新規参加者が案内動画を視聴
2. 「オープンチケット」をタップ
3. 個別チャンネルが作成される
4. 担当販売者（ロールで判別）が対応
5. 見積もり→決済→発送
```

### ■ 案内動画を見ていない場合

```
・テンプレートで「まず動画を見てください」と案内
・動画視聴後に再度チケットを開封してもらう
```

---

## 5. デリバリーレビュー

### ■ 目的

```
・取引実績を可視化
・新規顧客への信頼構築
・コミュニティの活性化
```

### ■ 運用ルール

| 項目 | ルール |
|------|--------|
| 投稿タイミング | 顧客の商品到着後 |
| 依頼方法 | 販売者から顧客にお願いする |
| フォーマット | 特に指定なし（自由形式） |
| 投稿チャンネル | #デリバリーレビュー |

### ■ 依頼テンプレ

```
商品の到着おめでとうございます！🎉

もしよければ、#デリバリーレビュー チャンネルに
感想や写真を投稿していただけると嬉しいです！

他のお客様の参考にもなりますし、
コミュニティの信頼性向上にも繋がります🙏
```

---

## 6. 処分セール

### ■ 概要

```
・在庫回転を上げるためのセール
・週1回以上の開催を想定
・開催タイミングは都度アナウンス
```

### ■ 告知方法

| 方法 | 詳細 |
|------|------|
| 全体アナウンス | #お知らせ チャンネルで告知 |
| 個別連絡 | 担当顧客にDMで案内 |

### ■ 価格設定

```
・処分対象商品は相場より安い前提
・さらに値下げして売り切る意識
・利益が薄くても回転を優先
```

---

## 7. チャンネル構成（推奨）

| チャンネル | 用途 |
|-----------|------|
| #ルール・案内 | サーバールール、案内動画 |
| #お知らせ | 新着、セール、重要告知 |
| #価格表 | 最新の価格リスト |
| #デリバリーレビュー | 顧客の到着報告 |
| #質問・雑談 | 自由なやり取り |
| （チケットチャンネル） | 個別対応（自動生成） |

---

## 8. 販売者の役割

### ■ 日常業務

| 業務 | 頻度 |
|------|------|
| 担当顧客への価格更新連絡 | 毎日（N&U更新後） |
| チケット対応 | 随時（即レス目標） |
| デリバリーレビュー依頼 | 商品到着後 |
| 処分セールの個別案内 | セール開催時 |

### ■ 担当顧客数の目安

| 項目 | 目安 |
|------|------|
| 1人あたり | 約400名 |
| 全員がVIPではないので対応可能 | |

---

## 9. まとめ：運用のポイント

1. **参加→動画視聴→チケット開封**の流れを徹底
2. **ロール付与で担当を明確に**（招待元 or ランダム）
3. **デリバリーレビューで信頼構築**
4. **処分セールで回転を上げる**
5. **即レスを意識**（早いほど成約率UP）
