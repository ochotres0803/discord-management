# SNS更新マニュアル

> 最終更新：2026年2月25日

---

## 1. 使用するSNS

| SNS | 優先度 | 備考 |
|-----|--------|------|
| **Instagram** | メイン | 主要な集客導線 |
| Facebook | サブ | 権限の問題で2名しか投稿できない |

---

## 2. 投稿の3段階

### ■ レベル1：簡単投稿（在庫写真）

| 項目 | 内容 |
|------|------|
| 内容 | 在庫写真、商品画像 |
| 特徴 | タイムスタンプを入れる |
| 素材 | N&Uの現場スタッフから共有 |
| 目的 | 日常的な更新、活動アピール |
| 頻度 | **毎日最低1回** |

### ■ レベル2：インプレッション稼ぎ

| 項目 | 内容 |
|------|------|
| 内容 | 開封動画、ARバルク、インパクト強めのコンテンツ |
| 特徴 | 編集にこだわる |
| 目的 | 認知拡大、インプレッション獲得 |
| 頻度 | **週3回以上** |

### ■ レベル3：信頼性訴求

| 項目 | 内容 |
|------|------|
| 内容 | 顔出しでの喋り投稿 |
| 特徴 | 信頼性を訴える、売りに行く構成 |
| 目的 | 信頼構築、Discord誘導 |
| 頻度 | **週3回の中に含める** |

---

## 3. 投稿頻度

| 種類 | 頻度 | 備考 |
|------|------|------|
| 簡単投稿（在庫写真） | **毎日1回以上** | 忙しくても可能 |
| 作り込んだ投稿 | **週3回以上** | 素材・編集にこだわる |

### ■ 週間スケジュール例

| 曜日 | 投稿内容 |
|------|----------|
| 月 | 在庫写真 |
| 火 | **作り込み投稿**（開封動画） |
| 水 | 在庫写真 |
| 木 | **作り込み投稿**（顔出し喋り） |
| 金 | 在庫写真 |
| 土 | **作り込み投稿**（信頼性訴求） |
| 日 | 在庫写真 or 休み |

## 4. コンテンツカテゴリ

| カテゴリ | 内容 | 目的 |
|----------|------|------|
| **セール** | 処分セール、特価案内 | 購買促進 |
| **在庫** | 新着、在庫写真 | 活動アピール |
| **実績紹介** | デリバリー報告、顧客の声 | 信頼構築 |
| **信頼性訴求** | 顔出し喋り、会社紹介 | 信頼構築 |
| **販売の仕方提案** | 購入方法、使い方 | 教育・誘導 |

### ■ 効果測定

```
・実行しながら実数値を計測
・効果が高いコンテンツに寄せていく
```

---

## 5. 素材について

### ■ 在庫写真（レベル1）

```
・N&Uの現場スタッフに撮影を依頼
・タイムスタンプを必ず入れる
・毎日共有してもらう
```

### ■ 作り込み投稿（レベル2・3）

```
・動画編集など専門性が高い作業 → 外注
・簡単な文字入れなど → 自分で行う
・素材の撮影と編集がボトルネック → 計画的に準備
```

---

## 6. ハッシュタグ

### ■ 方針

```
・市場の母数が大きすぎてハッシュタグで1位は難しい
・独自のハッシュタグを作成して目印にする
```

### ■ 推奨ハッシュタグ例

```
#[ブランド名]
#JapanTCG
#PokemonCardsJapan
#TradingCardsFromJapan
#[独自タグ] ← 作成推奨
```

---

## 7. 流入導線

### ■ プロフィール設計

```
プロフィール
 └ サイト欄に招待リンクを設置
    └ 各販売者ごとに計測可能な招待リンク
    └ mee6のInvite機能で作成
```

### ■ 投稿からの誘導

```
・投稿の最後に必ずDiscord誘導を入れる
・「詳しくはプロフィールのリンクから」
```

---

## 8. 担当者

| 投稿内容 | 担当 |
|----------|------|
| 日常投稿（在庫写真） | 各販売者 |
| 作り込み投稿 | 山崎 or 担当者 |
| 素材提供 | N&U現場スタッフ |
| 動画編集 | 外注 |

### ■ 流入後の担当

```
・プロフィールの招待リンクから流入
・リンクを発行した販売者が担当
・mee6で計測可能
```
