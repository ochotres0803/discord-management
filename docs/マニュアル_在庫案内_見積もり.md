# 在庫案内・見積もりマニュアル

> 最終更新：2026年2月25日

---

## 1. 在庫案内のタイミング

### ■ 1日のスケジュール

| 時間帯 | 内容 | 詳細 |
|--------|------|------|
| **前日夜** | メイン商材の価格紹介 | 売れ筋商品を先行案内 |
| | | 在庫整理完了前の仮案内 |
| **夕方** | 全種類の価格更新 | 買取が落ち着き、在庫整理完了後 |

### ■ 補足ルール

```
・価格更新されていない商品 → 問い合わせがあれば都度確認
・1日2回以上の案内を基本とする
```

---

## 2. 案内の優先順位

### ■ 優先して案内する商品

| 優先度 | 商品の特徴 |
|--------|-----------|
| **1位** | 在庫数が多いもの（＝需要が強く買取強化中） |
| **2位** | 再販の案内があった商品 |
| **3位** | 価格下落が予想される在庫 |
| **4位** | N&Uから「捌いてほしい」と指定された商品 |

### ■ 処分対象商品の扱い

```
・N&Uから指定された商品は、相場より安い前提
・さらに値下げして案内し、売り切る意識
・状況によって都度アナウンスあり
```

## 3. 案内チャネル

| チャネル | 用途 |
|----------|------|
| **Discord全体アナウンス** | 新着・セール・価格更新の告知 |
| **チケット（個別チャット）** | 担当顧客への個別案内 |

---

## 4. N&U以外のサプライヤー

### ■ 在庫がない場合の対応

```
1. N&Uに在庫がない場合
2. 社内の売買グループで他サプライヤーに確認
3. 在庫があれば、サプライヤーに発送代行を依頼
4. 請求書をもらい、山崎に報告
5. 振り込み → 振り込み履歴を報告
```

## 5. 見積もりのルール

### ■ フォーマット（統一）

```
【見積もり】
━━━━━━━━━━━━━━━━
商品名：[商品名]
数量：[個数]
単価：$[価格]
小計：$[合計]
━━━━━━━━━━━━━━━━
送料：$50（一律）
━━━━━━━━━━━━━━━━
合計：$[総額]
━━━━━━━━━━━━━━━━
有効期限：[明日の日付]まで
決済方法：Wise推奨（手数料0）
※PayPalは5%手数料がかかります
```

### ■ 価格決定のルール

| 項目 | ルール |
|------|--------|
| 基本価格 | N&Uが提示 |
| 価格交渉 | 販売者目線で売れない場合は相談OK |
| 値下げ判断 | 各自の判断でOK（粗利率6.5%目標を意識） |
| 報酬 | 粗利の20% → 安くしすぎると自分の報酬も減る |

### ■ 還付金について

```
・消費税還付金があるため、仕入れ値より安く売ることも可能
・これにより顧客の購入を促進し、回転数を上げる
```

---

## 6. 送料ルール

| 項目 | ルール |
|------|--------|
| 送料 | **一律5,000円（約$50）** |
| 理由（顧客目線） | お得、明朗会計で注文しやすい |
| 理由（販売者目線） | 提案しやすい |

### ■ 送料の意図

```
・顧客にはまず注文してほしい
・販売者にはまず販売体験を積んでほしい
・複雑な計算をなくしてハードルを下げる
```

---

## 7. 見積もり有効期限

| 項目 | ルール |
|------|--------|
| 有効期限 | **翌日中まで** |
| 理由 | 相場変動が毎日あるため |

### ■ 補足

```
・関係性次第で柔軟に対応してもOK
・ただし事故防止のため、翌日中が限界
・期限切れ後は再見積もりが必要
```

---

## 8. 決済方法の案内

### ■ 推奨順位

| 順位 | 方法 | 手数料 | 備考 |
|------|------|--------|------|
| **1位** | Wise | **0%** | 最推奨 |
| 2位 | PayPal | 5% | 手数料がかかる旨を伝える |

### ■ 案内テンプレ

```
【決済方法】
Wiseを推奨しています（手数料無料）
PayPalも可能ですが、5%の手数料がかかります

Wise送金先：[送金先情報]
PayPal：[PayPalアドレス]
```

---

## 9. 日次連絡ルール

### ■ 価格表更新時の連絡

```
・N&Uが価格表を更新したら、担当顧客に連絡
・早ければ早いほど成約機会が増える
```

### ■ 連絡の基準

| 対象 | 連絡 |
|------|------|
| アクティブな顧客 | 毎日連絡 |
| 1ヶ月以上返信なし | 連絡停止OK |
| 再度反応があれば | 再開 |

## 10. チェックリスト

見積もり作成時に確認：

- [ ] 商品名・数量が正確
- [ ] 価格が最新の価格表と一致
- [ ] 送料5,000円が含まれている
- [ ] 有効期限（翌日中）を記載
- [ ] 決済方法（Wise推奨）を案内
- [ ] PayPalの場合は5%手数料を伝えた
