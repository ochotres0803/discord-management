# 案内動画構成案

> 最終更新：2026年2月25日

---

## 1. 動画の基本情報

| 項目 | 内容 |
|------|------|
| **長さ** | 約1分 |
| **言語** | 英語 |
| **顔出し** | なし |
| **トーン** | カジュアル（やりすぎない） |

---

## 2. 動画の目的

```
・新規参加者に「注文の流れ」を理解してもらう
・チケット開封のハードルを下げる
・よくある質問を先に解消して対応工数を減らす
```

---

## 3. 構成案（約60秒）

### ■ タイムライン

| 秒数 | パート | 内容 | 画面イメージ |
|------|--------|------|-------------|
| 0-5秒 | **オープニング** | Welcome / サービス名 | ロゴ + タイトル |
| 5-15秒 | **自己紹介・サービス紹介** | 何をしている会社か | テキスト + 商品画像 |
| 15-25秒 | **特徴（強み）** | 安い・早い・品質 | アイコン + キーワード |
| 25-40秒 | **注文の流れ** | 見積もり→決済→発送 | ステップ図解 |
| 40-50秒 | **決済・送料** | Wise推奨、送料一律$50 | 料金表示 |
| 50-55秒 | **FAQ** | 何日で届く？→3-5日 | Q&A形式 |
| 55-60秒 | **CTA** | チケットを開いてね | ボタン誘導画面 |

## 5. 編集のポイント

### ■ トーン

```
・カジュアルだけどやりすぎない
・信頼感は大事にする
・ポップすぎるエフェクトは避ける
```

### ■ BGM

```
・アップテンポすぎない
・落ち着いたBGM or コーポレート系
・著作権フリーのものを使用
```

### ■ テロップ

```
・ナレーションの要点をテキスト表示
・英語で統一
・フォントはシンプル（読みやすさ重視）
```

## 6. 素材リスト

| 素材 | 用途 | 備考 |
|------|------|------|
| ロゴ | オープニング | 必要であれば作成 |
| 商品写真 | サービス紹介 | N&Uから提供 |
| 倉庫写真 | 信頼感演出 | N&Uから提供 |
| ステップ図解 | 注文フロー | デザイン作成 |
| 決済ロゴ | Wise, PayPal | 公式素材使用 |
| BGM | 全体 | 著作権フリー |
| ナレーション | 全体 | AI or 外注 |

---

## 7. 制作フロー

### ■ ステップ

```
1. 素材収集（写真、ロゴ）
2. スクリプト確定（上記を調整）
3. ナレーション収録 or AI生成
4. 図解・テロップ作成
5. 編集・BGM合成
6. レビュー・修正
7. 完成・Discord設置
```

### ■ 外注する場合

```
・動画編集者に依頼
・スクリプトと素材を渡す
・修正は1-2回で完了を目指す
```

---

## 8. 設置場所

| 場所 | 方法 |
|------|------|
| Discord | #ルール・案内 チャンネルに固定 |
| SNS | プロフィールにリンク |
| YouTube | 限定公開でアップロード |

### ■ Discordでの運用

```
・新規参加者は必ず動画を見てからチケットを開く
・動画を見ていない人には「まず動画を見て」と案内
```
