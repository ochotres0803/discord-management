# 販売ルール（日次連絡）

> 最終更新：2026年2月25日

---

## 1. 日次連絡の基本

### ■ トリガー

```
N&Uが価格表を更新したら → 担当顧客に連絡
```

### ■ タイミング

| 時間帯 | 内容 |
|--------|------|
| 前日夜 | メイン商材（売れ筋）の価格案内 |
| 夕方 | 全種類の価格更新後に連絡 |

### ■ 重要ポイント

```
・自分が担当する顧客には「すぐ」伝える
・早ければ早いほど成約機会が増える
・競合より先に連絡することが重要
```

---

## 2. 連絡対象

### ■ 顧客ランク別

| ランク | 連絡頻度 | 備考 |
|--------|----------|------|
| **VVIP**（月商100万超） | 毎日 | 最優先 |
| **VIP**（月商50万超） | 毎日 | 優先 |
| **Active** | 反応に応じて | 興味がある商品を中心に |
| **New** | 最初の数回 | 購入促進 |

### ■ 連絡停止の基準

| 状況 | 対応 |
|------|------|
| 1ヶ月以上返信なし | 連絡停止OK |
| 再度反応があれば | 再開 |

---

## 3. 連絡内容

### ■ 基本フォーマット

```
Hi [顧客名]! 

📦 New price list is up!

Today's highlights:
・[商品名] - $XXX
・[商品名] - $XXX
・[商品名] - $XXX

Let me know if you're interested!
Full list available in #price-list 📋
```

### ■ 顧客の興味に合わせる

```
・過去の購入履歴を参考に
・興味がありそうな商品を優先的に案内
・全員に同じ内容を送るのではなく、パーソナライズ
```

---

## 4. 処分セール時の連絡

### ■ セール告知

```
🔥 CLEARANCE SALE! 🔥

We need to move these items fast:
・[商品名] - $XXX (was $YYY)
・[商品名] - $XXX (was $YYY)

Limited stock, first come first served!
```

### ■ 優先商品

| 優先度 | 商品 |
|--------|------|
| 1 | N&Uから「捌いてほしい」と指定された商品 |
| 2 | 在庫数が多い商品（回転を上げたい） |
| 3 | 価格下落が予想される商品 |

## 5. 回答すべき質問

### ■ よくある質問

| 質問 | 回答例 |
|------|--------|
| **何日で届く？** | 3-5 business days with DHL/FedEx |
| 送料はいくら？ | Flat rate $50 (about ¥5,000) |
| 決済方法は？ | Wise recommended (0% fee), PayPal available (5% fee) |
| 価格交渉できる？ | Let me check, I'll get back to you |

## 6. 価格決定のルール

### ■ 基本

| 項目 | ルール |
|------|--------|
| 基本価格 | N&Uが提示 |
| 値下げ交渉 | 各自判断でOK |
| 目標粗利率 | 6.5% |
| 報酬 | 粗利の20% |

### ■ 判断基準

```
・還付金（10%）があるので、仕入れ値より安くてもOK
・顧客の購入を促進して回転数を上げる
・ただし利益ゼロは基本NG（意図があればOK）
```

## 7. 1日の業務フロー

### ■ 朝〜昼

```
□ 前日夜の問い合わせに返信
□ チケット対応
□ 個別チャットでの見積もり対応
```

### ■ 夕方（価格更新後）

```
□ 新しい価格表を確認
□ 担当顧客に価格更新を連絡
□ 処分対象商品を優先的に案内
```

### ■ 夜

```
□ 問い合わせ対応（可能な範囲で）
□ 翌日の素材準備（SNS用）
```

## 8. KPI意識

### ■ 個人で意識する指標

| 指標 | 目標 |
|------|------|
| 月商 | まず1,000万、目標1,500万 |
| VIP顧客数 | 増やす |
| リピート率 | 高める |
| 返信速度 | 早いほど良い |

### ■ 報酬計算

```
報酬 = 粗利 × 20%

例）月商1,500万、粗利率6.5%の場合
粗利 = 1,500万 × 6.5% = 97.5万
報酬 = 97.5万 × 20% = 19.5万
```
