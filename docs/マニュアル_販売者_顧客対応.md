# 販売者マニュアル：顧客対応全般

> 最終更新：2026年2月25日

---

## 1. 基本姿勢

### ■ レスポンス

| 項目 | ルール |
|------|--------|
| 目標 | **即レス**（努力目標） |
| 実運用 | なるべく早く |
| 理由 | 返信が早いほど成約しやすい。顧客も「やり取りが早い・楽」で常連化しやすい |

### ■ 対応時間

| 項目 | ルール |
|------|--------|
| 対応時間 | **実質24時間** |
| 理由 | 海外顧客との時差があるため |

※夜間対応は強制ではないが、早ければ早いほど機会損失を防げる

---

## 2. NGルール

### ■ NGワード

| NG | 理由 |
|----|------|
| 人種差別的な発言 | 絶対NG |
| 相手を否定する言葉 | 信頼関係を壊す |

### ■ NG対応

| NG | 理由 |
|----|------|
| **アンダーバリュー**（実際より低い価格での申告） | 違法行為 |
| **ギフト扱いでの送付** | 税関リスク |

※顧客から依頼されても断ること

---

## 3. 価格・利益について

### ■ 基本方針

```
・利益が全く残らない取引は基本NG
・ただし、顧客を育てる意図があり、会話の上でならOK
・各自で利益計算を行い、判断すること
```

### ■ 判断の目安

| 状況 | 判断 |
|------|------|
| 初回取引で信頼構築のため薄利 | ✓ OK |
| 大口顧客への継続取引で薄利 | ✓ OK |
| 何の意図もなく利益ゼロ | ✗ NG |

※報酬が粗利の20%なので、安くしすぎると自分の報酬も減る

---

## 4. クレーム対応

### ■ 基本姿勢

```
・クレームは早めに相談
・コミュニティでのサービス提供なのでブランディングを守る
・独断で大きな対応をしない
```

### ■ 対応権限

| 対応 | 権限 |
|------|------|
| 「次回値引きするよ」程度 | **各自判断OK** |
| 返金・返品対応 | **要相談** |
| トラブル拡大の可能性がある場合 | **即相談** |

---

## 5. コミュニケーションのトーン

### ■ 基本方針

```
・翻訳やAIを使うので、顧客に合わせて柔軟に
・敬語、カジュアル、絵文字 → 全てOK
・NGライン：失礼、取引相手として信用できない、と思われる対応
```

### ■ 言語

| 言語 | 対応 |
|------|------|
| 英語 | メイン（翻訳ツール活用） |
| その他言語 | 顧客に合わせて対応 |

---

## 6. 担当顧客の管理

### ■ 担当数の目安

| 項目 | 目安 |
|------|------|
| 1人あたり担当顧客数 | **約400名** |
| 全員がVIPではないので対応可能 | |

### ■ 連絡停止の基準

| 状況 | 対応 |
|------|------|
| 1ヶ月以上返信なし | 連絡停止OK |
| 再度反応があれば | 再開 |

---

## 7. まとめ：心がけること

1. **即レスを意識**（早いほど成約率UP）
2. **NGルールは厳守**（アンダーバリュー、ギフト扱いは絶対NG）
3. **クレームは早めに相談**（ブランディングを守る）
4. **顧客を育てる意識**（薄利でも意図があればOK）
5. **トーンは柔軟に**（失礼でなければOK）
