## 6. 計画の前提と性質

### ■ 悲観ベース設計

| 項目 | 計画値（悲観） | 実際の想定 |
|------|---------------|-----------|
| 山崎＋弟 | 4,000万/月 | 5,000万〜6,000万+ |
| 他メンバー | 計画値通り | 顧客成長で上振れ |
| 月商MAX | 4億 | 4.5億〜5億も視野 |

### ■ 達成ロジック

```
【顧客成長による売上拡大】

Q1-Q2：顧客獲得フェーズ
  └ 新規成約100名/月 × 6ヶ月 = 600名の顧客基盤
  
Q3-Q4：顧客成長フェーズ
  └ 新規→Active→VIP→VVIPへ昇格
  └ 1件あたり取引単価が上昇
  └ 件数は変わらなくても売上が倍増
```

## 7. 資金・収益構造

### ■ 資金の流れ

```
アクシスHD ──貸付──▶ クレイ ──デポジット──▶ N&U
                              （2,000万円）
                                   │
                              在庫（4,000万円）
                                   │
                              高回転で循環
                              （月8-10回転）
```

### ■ 収益構造

```
売上（100%）
  │
  └─ 原価（N&U卸価格）
       │
       └─ 粗利（6.5%）+ 還付金（10%）
            │
            └─ 共同事業コスト控除
                 ├─ 販売者成果報酬（粗利20%）
                 ├─ 発送関連変動費
                 ├─ クレイ側販管費（月額140万円）
                 ├─ N&U側販管費（月額80万円）
                 └─ 広告費等
                      │
                      └─ 利益分配（50:50）
```

## 8. リスクと対策

| リスク | 対策 | 備考 |
|--------|------|------|
| キャンセル | 先入金スタイル | リスクなし |
| 為替変動 | ヘッジ組み込み済み | 請求に反映 |
| クレーム | 1%以下・次回値引き対応 | 変動費に組込 |
| N&U調達能力 | 月商10億まで対応可 | 確認済み |
| N&U倒産 | 契約書でリスクヘッジ | 法務確認中 |
| 滞留在庫 | N&U国内販売で処分 | 実質返品運用 |

## 10. 成功の鍵

### ■ 3つの必達事項

1. **顧客エスカレーターの稼働**
   - 新規→Active→VIP→VVIPの昇格を確実に
   - リピートサイクル10日以内を維持

2. **組織の早期確立**
   - Q2までに14名体制完成
   - クレイリーダー候補の育成

3. **N&Uとの連携安定化**
   - 在庫消化48時間以内70%
   - 翌営業日出荷の維持
