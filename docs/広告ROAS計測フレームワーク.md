## 計測ルール

### 対象

**初回・新規取引のみ**を集計対象とする

| 対象 | 集計 |
|------|------|
| 新規顧客の初回取引 | ⭕ 対象 |
| 既存顧客のリピート取引 | ❌ 対象外 |

### 計算式

```
広告ROAS = 初回取引総額 ÷ 広告費
```

```
CAC（顧客獲得単価） = 広告費 ÷ 新規顧客数
```

## 3ヶ月累計での評価

リピート率50%を考慮し、3ヶ月スパンで評価する

### 3ヶ月評価シート

| 項目 | 計算式 |
|------|--------|
| 3ヶ月広告費合計 | 1ヶ月目 + 2ヶ月目 + 3ヶ月目 |
| 3ヶ月新規顧客数 | 1ヶ月目 + 2ヶ月目 + 3ヶ月目 |
| 3ヶ月初回取引総額 | 1ヶ月目 + 2ヶ月目 + 3ヶ月目 |
| 1ヶ月目顧客のリピート売上 | 1ヶ月目新規 × リピート率 × 平均単価 × 2回分 |
| 2ヶ月目顧客のリピート売上 | 2ヶ月目新規 × リピート率 × 平均単価 × 1回分 |
| **3ヶ月総売上** | 初回取引総額 + リピート売上合計 |
| **3ヶ月CAC** | 広告費合計 ÷ 新規顧客数 |
| **3ヶ月ROAS** | 3ヶ月総売上 ÷ 広告費合計 |

## 新規顧客の判別方法

### 方法案

1. **招待リンク別集計**
   - 各媒体・アカウントごとに招待リンクを発行
   - Discordの流入元で判別

2. **初回注文フラグ**
   - スプレッドシートで顧客ごとに「初回取引日」を記録
   - 初回かどうかを自動判定

3. **アンケート**
   - Discord参加時に「どこで知りましたか？」を聞く

- Q1（1-3月）、Q2（4-6月）、Q3（7-9月）、Q4（10-12月）
